API Documentation

Authentication

Include your API key in the X-API-Key header:

curl -H "X-API-Key: YOUR_API_KEY" https://api.vex.app/api/public/v1/projects

Get Projects

Retrieve all your projects:

GET /api/public/v1/projects

Get Files in Project

Retrieve all files in a project:

GET /api/public/v1/projects/{project_id}/files

Create Project

Create a new project:

POST /api/public/v1/projects
