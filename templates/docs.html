Vex API Documentation

Vex is a modern note-taking and productivity engine. Use the Vex API to programmatically extract notes, input tasks, perform CRUD operations, and seamlessly bridge your Google Workspace with our AI backend.

Get Started arrow_downward

Authentication

The Vex API uses JSON Web Tokens (JWT) to authenticate requests. You can generate an API key from your Developer Dashboard after logging in via Google or GitHub. All API requests must be made over HTTPS and include your token in the Authorization header.

cURL Request 
curl -X GET https://api.vex.io/v1/user \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json"

Notes (CRUD)

Create, read, update, and delete notes within your workspace. Notes support markdown and link directly into your Neural Graph.

POST /api/v1/notes

Creates a new note in your default workspace.

Request Body
{
  "title": "Physics - Thermodynamics",
  "content": "The first law of...",
  "tags": ["jee", "physics"],
  "folder_id": null
}
Response 200 OK 
{
  "id": "nt_8f92j1kx",
  "status": "created",
  "created_at": "2024-10-24T10:00:00Z",
  "graph_links": 0
}

Google Workspace Integration

Once a user links their Google Account via OAuth, the Vex API acts as a unified proxy to fetch events, manage tasks, and summarize emails using the AI backend.

GET /api/v1/workspace/calendar/upcoming

Fetches the authenticated user's Google Calendar events for the next 7 days, summarized by Vex AI.

Response Example
{
  "sync_status": "active",
  "events": [
    {
      "event_id": "gcal_9x8z",
      "title": "Product Sync",
      "start_time": "2024-10-25T09:00:00Z",
      "ai_context": "Related to your 'Project Alpha' notes."
    }
  ]
}
